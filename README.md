# Sales and Marketing Reporting CNI

The code in this directory represents a [low-code](https://prismatic.io/docs/integrations/low-code-integration-designer/) integration that has been [converted](https://prismatic.io/docs/integrations/code-native/get-started/convert-low-code-code-native/) to its [code-native](https://prismatic.io/docs/integrations/code-native/) equivalent.
By converting a low-code integration to code-native, you can take advantage of the full power of TypeScript and the NPM ecosystem as you build your integration.

> Please note that conversion is a one-way operation. Once you convert a low-code integration to code-native, you cannot convert it back to low-code.

## Post-export steps

After exporting your low-code integration to code-native, there are several steps you should take to ensure that you're taking advantage of TypeScript and the NPM ecosystem, including removing unused step result assignments, simplifying branch steps, adding type annotations, and simplifying code steps.
For a detailed guide on these steps, please refer to the [post-export steps documentation](https://prismatic.io/docs/integrations/code-native/get-started/convert-low-code-code-native/#post-generation-instructions).
